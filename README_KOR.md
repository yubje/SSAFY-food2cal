## 💻  Technology Stack

#### 📌 Frontend: Vue.js

#### 📌 AI: Flask

#### 📌 Image Storage: AWS S3


### Core Technology  

**Flask**: food2cal 의 머신러닝 기능 구현

**YOLO v2**: 머신러닝 기능 구현

**Vue.js**: food2cal의 프론트엔드 기능 구현

**Axios**: http 통신 라이브러리, 비동기 통신 요청 처리

**AWS**: EC2 서비스를 이용하여 Ubuntu 서버 구축, S3 서비스를 이용하여 이미지 스토리지 구축
